# Avatar

Soft-blue initials chip used in the app rail and profile menus. Pass `src` for a real photo.

```jsx
<Avatar initials="MG" />
<Avatar initials="MG" size="xl" />
<Avatar src="/me.jpg" alt="Maya" size="lg" />
```
`size`: `sm` 28 · `md` 34 · `lg` 44 · `xl` 52, or a number. `onBrand` brightens the fill for the blue sidebar.
