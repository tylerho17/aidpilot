# NavItem

Sidebar nav row for the AidPilot blue rail. Active = solid white pill with blue text/icon; inactive = translucent white. `on="light"` reuses it on white surfaces; `tag` adds a trailing label like "Soon".

```jsx
<NavItem icon="grid" label="Dashboard" active />
<NavItem icon="shield" label="Protect Aid" />
<NavItem icon="star" label="Schools" tag="Soon" />
```
