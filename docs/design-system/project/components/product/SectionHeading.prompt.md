# SectionHeading

Colored uppercase eyebrow over a Nunito-900 title, with optional subtitle and a trailing action. `align="center"` + `size="lg"` for marketing section heads; `size="sm"` left-aligned for in-app page titles.

```jsx
<SectionHeading eyebrow="The product" align="center" size="lg"
  title="One place to know what needs attention." />
<SectionHeading title="Your scholarships" size="sm"
  action={<Button size="sm">Refresh matches</Button>} />
```
