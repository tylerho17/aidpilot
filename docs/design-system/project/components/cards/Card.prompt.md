# Card

The base AidPilot surface — white, hairline blue-tinted border, 20px radius, soft blue-tinted shadow. Nearly everything in the product sits on one.

```jsx
<Card lift padding={24}>…</Card>
<Card as="a" href="/fafsa" lift>…</Card>
```
Set `lift` for the signature hover raise. `padding`, `radius`, and `as` are adjustable.
