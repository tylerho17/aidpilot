# StatCard

Dashboard metric: a small muted label over a big Nunito-900 number. Lifts on hover.

```jsx
<StatCard label="Aid secured this year" value="$18,400" sub="+$2,100 vs last check" />
<StatCard label="At risk" value="$0" valueColor="var(--green-600)" />
```
Color the `value` to carry meaning (green secured, coral at-risk).
