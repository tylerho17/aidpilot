# TextField

Rounded text input. The 1.5px border turns brand-blue with a soft focus ring; `error` turns it coral and shows the message. Optional `label`, leading `icon`, and `hint`.

```jsx
<TextField label="Email" type="email" placeholder="you@school.edu" />
<TextField icon="star" placeholder="Search scholarships" />
<TextField label="GPA" error="Enter a number between 0 and 4.0" />
```
